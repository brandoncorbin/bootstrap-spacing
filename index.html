Bootstrap Spacing Examples

These classes are to help give nice spacing and whitespace to bootstrap elements.
It's goal is to help deveopers stop using .row and .col-xs-12 for the spacing they desire.

spacer spacer-sm hairline hairline-top hairline-bottom padding padding-lg
spacer spacer-md hairline hairline-top hairline-bottom padding padding-xl
